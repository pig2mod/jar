# Pig2 mod for Minecraft Forge 1.20.1
The strongest pig that can win any mob battle in Minecraft 1.20.1

This Forge MOD adds a special pig "Pig2" to Minecraft.
If you summon Pig2 near a living entity, Pig2 will kill it, no matter how strongly modded it is.
Enjoy your mob battles with Pig2 and any other MOD mobs.
